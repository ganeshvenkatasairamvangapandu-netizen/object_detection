# Real-Time Object Detection using OpenCV & YOLO

## Overview
This project performs real-time object detection using a webcam feed. It detects objects like persons, phones, and bottles using a lightweight YOLOv3-tiny model.

## Features
- Real-time detection using webcam
- Lightweight and fast (YOLOv3-tiny)
- Easy to run on any system

## Tech Stack
- Python
- OpenCV
- NumPy
